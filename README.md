## Getting Started

### Prerequisites

- Python PIP
- npm

### Frontend

1. `cd frontend`
2. `npm i `
3. `npm start`

### Backend

1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn app.main:app --reload`

- Access localhost URL https://localhost:3000
